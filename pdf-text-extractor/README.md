# PDF 텍스트 추출기

무료이며 안전한 PDF 텍스트 변환 도구입니다.

## 특징

- 🔒 **100% 클라이언트 사이드 처리** - 파일이 서버로 전송되지 않아 안전합니다
- 📱 **반응형 디자인** - 모바일, 태블릿, 데스크톱 모든 기기에서 사용 가능
- 🚀 **빠른 처리** - PDF.js를 활용한 빠른 텍스트 추출
- 💰 **완전 무료** - 제한 없이 무료로 사용 가능

## 사용 기술

- HTML5
- Tailwind CSS (CDN)
- PDF.js v3.11.174 (CDN)

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. PDF 파일을 드래그 앤 드롭하거나 파일 선택 버튼 클릭
3. 추출된 텍스트 확인 및 복사

## 파일 구조

```
pdf-text-extractor/
├── index.html    # 메인 HTML 파일 (CSS, JS 내장)
├── README.md     # 프로젝트 설명
└── .gitignore    # Git 제외 파일 목록
```
