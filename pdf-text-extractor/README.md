# PDF 텍스트 추출기 (PDF Text Extractor)

브라우저에서 바로 작동하는 무료 PDF 텍스트 추출 도구입니다.

## 주요 기능

- **100% 클라이언트 사이드 처리**: PDF 파일이 서버로 전송되지 않고 사용자의 브라우저 내에서만 처리되어 개인정보 유출 걱정이 없습니다.
- **간편한 사용**: 별도의 설치 없이 웹 브라우저만 있으면 바로 사용 가능합니다.
- **반응형 디자인**: 모바일 및 데스크탑 환경 모두에서 최적화된 화면을 제공합니다.

## 기술 스택

- **HTML5**: 시맨틱 마크업 구조
- **Tailwind CSS (CDN)**: 빠르고 유연한 스타일링
- **PDF.js (v3.11.174)**: Mozilla의 PDF 리더 라이브러리를 사용한 텍스트 추출
- **Vanilla JavaScript**: 외부 프레임워크 의존성 없는 순수 자바스크립트 구현

## 설치 및 실행

이 프로젝트는 별도의 빌드 과정이 필요하지 않습니다.

1. 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
